Langkah pembuatan github page menggunakan sistem git

1. Jika sudah selesai membuat blog, silahkan membuka github di www.github.com. Lalu klik pada bagian create repository untuk membuka repository baru pada github

2. Kemudian isi repository name dengan : namauser.github.io , Jika sudah klik create pada bagian bawah

3. Lalu kita kembali ke folder web kita yang ingin ditambahkan ke github, lalu klik kanan dan pilih “Git Bash Here”

4. Kemudian membuat inisiasi repository menggunakan “git init”

5. Langkah berikutnya yaitu melakukan perintah “git clone” untuk melakukan cloning dari repository github ke local

6. Jika sudah, block keseluruhan file untuk websitenya dan dimasukkan ke file anisyaulfitriaa.github.io

7. Kemudian lakukan perintah “git add .” untuk menambahkan semua file dan directory yang akan di commit.

8. Lalu lakukan perintah “git commit –m ‘first commit’ ” karena ini merupakan commit pertama jadinya memberikan pesan berupa first commit

9. Langkah yang terakhir yaitu melakukan “git push –u origin master” untuk melakukan push file local ke dalam repository github

10. Lalu kita cek ke github dan merefresh-nya, jika tampilannya sudah seperti di bawah ini maka dapat dikatakan local git repository telah berhasil di clone ke Github repository.

11. Untuk memastikan yang terakhir, cek website dengan alamat username.github.io di browser. Bisa kita lihat pada gambar di bawah ini website sudah bisa diakses.
